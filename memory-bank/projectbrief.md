# Project Brief: Texas PLLC Formation Service

## Overview
A premium Texas Professional Limited Liability Company (PLLC) formation service positioned as the only PLLC-focused provider offering transparent pricing, profession-specific expertise, and a 48-hour filing guarantee.

## Core Value Proposition
- **$749 all-inclusive** pricing (transparent, no hidden fees)
- **48-hour filing guarantee** (or $100 credit if missed)
- **Profession-specific expertise** (Therapist, Physician, Engineer, Attorney)
- **Complete service** including state filing, EIN, Operating Agreement, Registered Agent, and compliance support

## Target Market
Professionals forming Texas PLLCs:
- Therapists/Counselors
- Physicians/Physician Assistants
- Engineers (PE)
- Attorneys

## Business Model
- **Base Package:** $749 (includes $300 state fee)
- **Add-ons:**
  - Board handling: +$200
  - Physician/PA joint entity: +$149
  - Foreign PLLC registration: $1,195
- **Unit Economics:** ~$386 net margin per order (at $150 CPA → $236 profit)

## Launch Goals (90 days)
- 150 paid orders
- Average CPA ≤ $150
- On-time filing ≥ 95%
- Refund rate ≤ 3%

## Technical Stack
- **Frontend/Backend:** Next.js
- **Hosting:** Vercel
- **Checkout:** spiffy.co embedded form
- **Analytics:** GA4 + Enhanced Conversions

## Key Differentiators
1. **Specialization:** Only PLLC-focused provider
2. **Transparency:** Clear pricing, no upsells
3. **Speed:** 48-hour filing guarantee
4. **Trust:** 60-day money-back guarantee
5. **Expertise:** Profession-specific templates and guidance

## Out of Scope (Phase 1)
- Multi-state formations
- Bookkeeping/tax filings
- Complex ownership (>10 owners)
- Webinars
